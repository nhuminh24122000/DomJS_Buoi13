Bài tập DOM JS

Bài tập 1: Tính tiền lương nhân viên

Sơ đồ 3 khối

  • Khối 1: Input
    sal (lương 1 ngày), numDay (số ngày làm)
  • Khối 2: Các bước xử lý
    • Tạo hàm, gắn sự kiện click Lấy giá trị từ form
    • Kiểm tra điều kiện và lập công thức tính lương nhân viên
      Kiểm tra điều kiện
      Nếu sal <= 0 hoặc numDay < 0 Thông báo nhập lại
      Lập công thức
      totalSalary = sal * numDay
    • Hiển thị kết quả
  • Khối 3: Output
    totalSalary

Bài tập 4: Tính diện tích, chu vi hình chữ nhật

Sơ đồ 3 khối

  • Khối 1: Input
    lengthRec (chiều dài hcn), breadthRec (chiều rộng hcn)
  • Khối 2: Các bước xử lý
    • Tạo hàm, gắn sự kiện click Lấy giá trị từ form
    • Kiểm tra điều kiện và lập công thức tính diện tích, chu vi hình chữ nhật
      Kiểm tra điều kiện
      Nếu lengthRec <= 0 hoặc breadthRec <= 0 Thông báo nhập lại
      Lập công thức
      Diện tích: area = lengthRec * breadthRec
      Chu vi: perimeter = (lengthRec + breadthRec) * 2
    • Hiển thị kết quả
  • Khối 3: Output
    area, perimeter

Bài tập 5: Tính tổng 2 ký số

Sơ đồ 3 khối

  • Khối 1: Input
    num (số nguyên dương có 2 chữ số)
  • Khối 2: Các bước xử lý
    • Tạo hàm, gắn sự kiện click Lấy giá trị từ form
    • Kiểm tra điều kiện và lập công thức tính tổng 2 ký số
      Kiểm tra điều kiện
      Nếu num >= 100 hoặc num < 10 hoặc Number.isInteger(num) == false
      Thông báo nhập lại
      Lập công thức
      totalDigits = Math.floor(num / 10) + (num % 10)
    • Hiển thị kết quả
  • Khối 3: Output
    totalDigits
